# JP PROD - Video Production Company Website

A modern, minimalist one-page website for JP PROD video production company. Built with Next.js, TypeScript, and Tailwind CSS.

## Project Structure

```
jpprod_webpage/
├── frontend/          # Next.js frontend application
└── backend/           # Backend (to be implemented)
```

## Features

- ✨ Modern black and white design
- 🎬 One-page scroller with smooth animations
- 📱 Fully responsive layout
- ⚡ Built with Next.js 16 and React
- 🎨 Styled with Tailwind CSS
- 📝 TypeScript for type safety
- 🚀 Optimized for performance

## Sections

1. **Hero** - Introduction with company tagline
2. **Media/Services** - Portfolio of video production services
3. **Companies** - Clients and partners showcase
4. **Contact** - Contact information and social media links

## Getting Started

### Frontend Development

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
cd frontend
npm run build
npm start
```

### Linting

```bash
cd frontend
npm run lint
```

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment Ready:** Vercel, Netlify, or any Node.js hosting
